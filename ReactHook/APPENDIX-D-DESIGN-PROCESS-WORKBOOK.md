
| 检查项 | 状态 | 备注 |

下面是一份填写好的示例（useOnlineStatus）：

```
Hook 名称: useOnlineStatus

问题描述:
我需要知道用户当前离线还是在线，并在网络状态变化时自动更新。

Step 1: 识别响应式值
1. isOnline (在线/离线状态) —— 会随着网络状态变化

Step 2: 分类
| 值       | 类别  | 为什么？                     |
| isOnline | State | UI 需要反映当前在线状态       |

Step 3: 契约
function useOnlineStatus(): boolean

Step 4: 原语
| isOnline | useState | 存储并触发渲染 |
依赖图: useEffect(监听事件) → setState(更新) → useState 返回新值

Step 5: 实现
（见 Module 3）

Step 6: 审查
- 清理: ✅ 事件监听器已正确移除
- deps: ✅ [] 正确（handler 不变）
- SSR: ✅ 使用 typeof navigator !== 'undefined' 保护
- 模式: Composer (useState + useEffect 组合)
```
