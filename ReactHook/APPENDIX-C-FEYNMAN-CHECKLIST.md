## Module 0: 第一性原理

- [ ] **T2**: 自定义 Hook 和普通 JavaScript 函数有什么区别？运行时和语法两个层面各有什么不同？

**自查**：听完自己的讲解——哪里卡壳了？哪里用了 jargon 但没有解释？哪里你自己都说不清楚？

## 费曼学习法检查标准

对于每个费曼检查点，用以下标准自评：

1. **清晰度**：解释中没有任何你无法进一步解释的术语吗？
2. **简洁度**：能在 3 分钟或 3 句话内讲清楚吗？
3. **类比质量**：能用生活类比（不是代码类比）解释吗？
4. **缺口识别**：解释中哪个部分你犹豫了？那里就是缺口。
