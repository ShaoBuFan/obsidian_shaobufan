## 实施顺序建议

```
MCP 服务器 ──→ 复合评分 ──→ 对话入库 ──→ 三层架构
   │               │             │             │
   │               │             │             └── 需要 LLM 批量调用
   │               │             └── 依赖 stop hook 配置
   │               └── 依赖 search_notes 先可用
   └── 入口点：让 Claude Code 能调向量记忆库
```

> MCP 服务器是瓶颈任务——没有它，复合评分和对话入库都无法通过 Claude Code 调用。所以建议第一个做。
