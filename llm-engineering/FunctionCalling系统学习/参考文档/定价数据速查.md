## 工具调用的典型成本（以 Sonnet 4.6 为例）

| 操作 | Token 消耗 | 成本 |
|------|-----------|------|
| 3 个工具定义（每工具 ~200 token） | ~600 input | ~$0.0018 |
| 单次工具调用（50 output token） | 50 output | ~$0.0008 |
| 工具结果中等大小（500 token） | 500 input（下轮） | ~$0.0015 |
| 工具结果较大（5000 token） | 5000 input（下轮） | ~$0.015 |

### 典型三轮工具调用：~$0.015
### 复杂多工具编排（20 轮）：$0.10-0.50
### PTC 一次性执行（3 工具）：相比传统节约 ~37% token
