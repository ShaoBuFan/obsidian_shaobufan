# 官方文档索引

## 基准测试

### Function Calling 质量
- **BFCL V4 (Berkeley Function-Calling Leaderboard)**：`gorilla.cs.berkeley.edu/leaderboard.html` — ICML 2025，五轴评估（单轮/多轮/Agent/幻觉/格式敏感性）
- **Live API-Bench**：`arxiv.org/abs/2506.11266` — 2500+ 真实 API，多步工具调用

### 延迟与性能
- **AutoBench**：`autobench.org` — 10+ 工具使用类别的延迟数据
- **infermark**：`pypi.org/project/infermark` — 开源延迟基准测试工具
- **bench-loop**：`github.com/outsourc-e/bench-loop` — 本地 Agent 循环基准测试

### 学术论文
- **"When Do Tools and Planning Help LLMs Think?"** (arXiv:2601.02663, Jan 2026) — 工具调用 40x 延迟增加，准确率 47.5%→67.5%
- **AsyncFC (UC Berkeley, Fall 2025)** — 异步工具调用 4.4% 加速，SWE-bench 验证
- **SMCP (arXiv:2602.01129, Feb 2026)** — MCP 安全加固（工具中毒、凭证窃取、权限提升）
- **"Prompts Don't Protect" (arXiv:2605.18414, May 2026)** — Prompt 级访问控制的根本性失败
