# Function Calling / Tool Use 系统学习

## 为什么研究这个主题

LLM 的本质能力是文本生成——给定 token 序列，预测下一个 token。但现实世界的任务往往需要 LLM 与外部系统交互：查询数据库、调用 API、执行代码、操作文件。==Function Calling==（OpenAI 术语）和 ==Tool Use==（Anthropic 术语）就是连接 LLM 的文本世界与外部工具世界的桥梁。

> 工具调用不是让 LLM "变聪明"的技术，而是让 LLM 的文本生成能力变得**可操作**的接口层。
