# ReAct 参考文档

## 原始论文

- **ReAct: Synergizing Reasoning and Acting in Language Models**
  - arXiv: [2210.03629](https://arxiv.org/abs/2210.03629)
  - 作者：Yao, Zhao, Yu, Du, Shafran, Narasimhan, Cao (Google Research)
  - ICLR 2023
  - 代码：[github.com/ysymyth/ReAct](https://github.com/ysymyth/ReAct)
  - 项目页：[react-lm.github.io](https://react-lm.github.io)
  - Google Research Blog：[research.google/blog/react-synergizing-reasoning-and-acting-in-language-models](https://research.google/blog/react-synergizing-reasoning-and-acting-in-language-models/)
  - HuggingFace Paper Page：[huggingface.co/papers/2210.03629](https://huggingface.co/papers/2210.03629)

## 后继架构

| 架构 | 论文 | 年份 | 核心创新 |
|------|------|------|---------|
| **Plan-and-Solve** | Wang et al., arXiv:2305.04091 | 2023 | 全局规划后再逐步执行 |
| **Reflexion** | Shinn et al., arXiv:2303.11366 | 2023 | 跨次尝试的自评学习 + episodical memory |
| **Tree-of-Thought** | Yao et al., arXiv:2305.10601 | 2023 | 多路径搜索 + 剪枝 + 回溯 |
| **LATS** | Zhou et al., arXiv:2310.04406 | 2023 | Tree-of-Thought + Reflexion 混合 |
| **Pre-Act** | Uniphore, arXiv:2505.09970 | 2025 | 前置规划阶段，内置合规嵌入 |
| **Select-then-Solve** | arXiv:2604.06753 | 2026 | 学习路由器，根据任务动态选择范式 |
| **Co-ReAct** | arXiv:2605.23590 | 2026 | 评分标准驱动的步级引导，GRPO 训练 |

## 2026 年演进

| 方向 | 代表 | 关键变化 |
|------|------|---------|
| 半形式化推理 | ReAct V3 (ServiceNow) | 强制逻辑证书，准确率 78%→93% |
| 并行执行 | ReAct V3 | 证据门替代完成门，延迟降低 ~40% |
| 评分标准引导 | Co-ReAct | 评分标准生成器作为即插即用组件 |
| 代码 Action | CodeAct | 从 API 调用到 Python 代码执行 |
| 模型内生 | GPT-5.x, Claude Opus 4.6+ | 通过 RL 训练获得，非 prompt 外挂 |
