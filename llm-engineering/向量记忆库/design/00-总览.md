# 向量记忆库 · 总览

## 这是什么

一个个人向量记忆系统，让 Claude Code 能够语义检索你的历史笔记和对话，基于你的过去提供有上下文的回答。

## 完整数据流

```
┌─────────────────────────────────────────────────────┐
│                    写入侧                            │
│                                                     │
│  Obsidian 笔记 ──→ 切分 ──→ embed ──┐              │
│                                      │              │
│  Claude Code 对话 ──→ 提取 ──→ embed ──┤           │
│                                      │              │
│  显式保存 ──→ embed ───────────────→ │              │
│                                      ▼              │
│                            ┌───────────────┐        │
│                            │  向量记忆库    │        │
│                            │               │        │
│                            │ ChromaDB      │        │
│                            │ + 复合评分     │        │
│                            └───────┬───────┘        │
│                                    │                │
│                    读取侧          │                │
│                                    ▼                │
│  Claude Code ──→ MCP Server ──→ search_notes ──→ 检索结果拼入上下文  │
│                                                     │
│                    维护侧                            │
│                                                     │
│  笔记变更 ──→ sync_notes ──→ 更新索引              │
│  孤儿清理 ──→ cleanup ──→ 删除失效 chunk           │
└─────────────────────────────────────────────────────┘
```
