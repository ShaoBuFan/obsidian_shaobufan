# CS 自学指南

基于 [csdiy.wiki](https://csdiy.wiki/) 整理的自学路线与课程索引。

> 源项目由 PKUFlyingPig 维护，收录了国内外顶尖高校（UC Berkeley、MIT、Stanford、CMU、Harvard 等）的 100+ 门开源免费课程。

## 核心学习路线

### 第一阶段：入门（约 3-4 个月）

1. **[UCB CS61A](https://csdiy.wiki/编程入门/Python/CS61A/)** — Python 编程入门，基于 SICP 改编
2. **[MIT-Missing-Semester](https://csdiy.wiki/编程入门/MIT-Missing-Semester/)** — 命令行、编辑器、版本控制等工程技能
3. **必学工具** — Git、Vim、Docker、LaTeX（边用边学）

### 第二阶段：数学与编程深化（约 4-6 个月）

4. **数学基础** — 微积分 MIT 18.01/02 → 线性代数 MIT 18.06 → 离散数学 UCB CS70
5. **[Harvard CS50](https://csdiy.wiki/编程入门/C/CS50/)** — C 语言与计算机科学基础
6. **[CMU 15-213 CSAPP](https://csdiy.wiki/计算机系统基础/CSAPP/)** — 计算机系统核心
7. **[UCB CS61B](https://csdiy.wiki/数据结构与算法/CS61B/)** — 数据结构与算法

### 第三阶段：系统核心（约 6-8 个月）

8. **体系结构** — UCB CS61C 或 Nand2Tetris
9. **操作系统** — [MIT 6.S081](https://csdiy.wiki/操作系统/MIT6.S081/) 或 UCB CS162
10. **计算机网络** — [Stanford CS144](https://csdiy.wiki/计算机网络/CS144/)
11. **数据库** — [CMU 15-445](https://csdiy.wiki/数据库系统/15445/)
12. **编译原理** — Stanford CS143 或 PKU 编译原理实践

### 第四阶段：方向深入（按兴趣选择）

- **并行与分布式** — CMU 15-418 / MIT 6.824
- **机器学习** — Stanford CS229 → CS231n / CS224n
- **深度学习** — 李宏毅 / CS230
- **计算机图形学** — GAMES101 → GAMES202
- **Web 开发** — Full Stack Open / Stanford CS142
- **系统安全** — UCB CS161 / MIT 6.858

## 使用建议

- **不要贪多** — 核心路线中的课程认真啃完比什么都强
- **做作业** — 只看视频不做作业 = 没学
- **用英文资料** — 翻译版二手信息损耗大，直接看原版
- **记录笔记** — 每门课在此 vault 中开一个笔记文件，用法和 `重构/` 一样
