# 体系结构

> 软件和硬件的交界处。理解 CPU 怎么执行你的代码。

## 课程

| 课程 | 学校 | 难度 | 说明 |
|------|------|------|------|
| **Coursera Nand2Tetris** | — | ⭐ | 从 NAND 门开始造一台计算机。适合零基础 |
| **UCB CS61C** | UCB | ⭐⭐⭐ | RISC-V、流水线、缓存、SIMD、并行 |
| **ETHz DDCA** | ETH Zurich | ⭐⭐⭐ | 数字设计与计算机体系结构 |
| **ETHz CA** | ETH Zurich | ⭐⭐⭐⭐ | 高级体系结构 |

## 学习路线

1. **Nand2Tetris**（Part 1）— 建立直觉：门电路 → ALU → CPU → 汇编器
2. **CS61C** — 系统学习：RISC-V ISA、流水线冒险、缓存一致性、虚拟内存的硬件视角
3. 进阶按需选择 ETHz 课程

## 核心概念

- ==指令集架构（ISA）== — 软件的硬件接口
- ==流水线== — 指令级并行
- ==缓存层次== — 时间局部性 + 空间局部性
- ==虚拟内存== — 每个进程有自己的地址空间
- ==超标量 / 乱序执行== — 现代 CPU 如何榨干性能
