# 计算机图形学

## 核心课程

| 课程 | 说明 |
|------|------|
| **GAMES101** | 闫令琪。入门首选。光栅化、光线追踪、几何、渲染方程 |
| **GAMES202** | 实时渲染：阴影、环境光照、PBR |
| **GAMES103** | 物理模拟：刚体、布料、流体 |

## 学习路线

1. **GAMES101** — 必学。用 C++ 写一个软渲染器
2. **GAMES202** — 实时渲染方向
3. **GAMES103** — 物理模拟方向

## 核心概念

- ==光栅化== — 三角形 → 像素
- ==光线追踪== — 像素 → 光线 → 场景求交
- ==渲染方程== — Kajiya 1986
- ==PBR== — 基于物理的材质模型
- ==几何处理== — 网格、曲面、隐式表达
