# Web 开发

## 课程

| 课程 | 学校 | 说明 |
|------|------|------|
| **MIT Web Development** | MIT | Web 开发入门 |
| **Stanford CS142** | Stanford | 全栈 Web 应用：前端、后端、数据库、安全 |
| **Full Stack Open** | Helsinki | React + Node.js + GraphQL。实践导向，最适合自学者 |
| **CS571 Building UI** | — | React & React Native |

## 学习建议

> Web 开发技术迭代极快。不要去系统学 HTML/CSS/JS 理论再开始写——直接跟 Full Stack Open 做项目，边做边学。基础知识（HTTP 协议、浏览器渲染、REST API）在做项目的过程中自然掌握。

## 核心技术栈

- **前端**：HTML、CSS、JavaScript → React / Vue
- **后端**：Node.js / Python Flask-Django / Go
- **数据库**：PostgreSQL / MongoDB
- **部署**：Docker、CI/CD
