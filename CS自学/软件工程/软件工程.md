# 软件工程

## 课程

| 课程 | 学校 | 说明 |
|------|------|------|
| **MIT 6.031: Software Construction** | MIT | 代码质量、测试、设计模式、并发编程。Java |
| **UCB CS169: Software Engineering** | UCB | SaaS、敏捷、TDD、Rails |
| **CMU 17-803: Empirical Methods** | CMU | 软件工程的实证研究 |

## 核心主题

- ==设计模式== — 常见问题的可复用解决方案
- ==测试== — 单元测试、集成测试、TDD
- ==重构== — 改善既有代码设计（与 `重构/` 中的笔记形成互补）
- ==敏捷开发== — Scrum、Kanban、持续交付
- ==代码审查== — Code Review 是发现 bug 最便宜的方式

> 软件工程的核心不是学框架，是理解怎么写出**能持续演化**的代码。配合 `重构/` 目录中《软件设计的哲学》笔记一起学习效果最好。
