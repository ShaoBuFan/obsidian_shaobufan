# 编程入门

## 核心推荐

| 课程 | 语言 | 说明 |
|------|------|------|
| **[UCB CS61A](https://csdiy.wiki/编程入门/Python/CS61A/)** | Python | 🔴 第一门编程课首选。基于 SICP 改编 |
| **[MIT-Missing-Semester](https://csdiy.wiki/编程入门/MIT-Missing-Semester/)** | 工具 | 🔴 配合任何编程课同步学 |
| **[Harvard CS50](https://csdiy.wiki/编程入门/C/CS50/)** | C | 🟡 学完 Python 后深入理解计算机底层 |

## 语言专项

### Python
- **UCB CS61A**：首选
- **CS50P**：Harvard 的纯 Python 课，更轻松
- **MIT 6.100L**：MIT 入门课

### C
- **Harvard CS50**：从 Scratch → C → Python，覆盖面广
- **Duke Introductory C Programming**：Coursera 专项课

### C++
- **Stanford CS106L**：标准 C++ 编程，推荐
- **Stanford CS106B/X**：数据结构 + C++

### Rust
- **Stanford CS110L**：系统编程安全
- **KAIST CS220 / CS431**：编程原理 / 并发编程

### 函数式编程
- **Cornell CS3110**：OCaml
- **Haskell MOOC**：Haskell

## 学习建议

> 第一门语言选 Python（CS61A），第二门选 C（CS50）。Python 让你快速建立编程思维，C 让你理解计算机在干什么。之后 C++/Rust/函数式语言按兴趣选择。
