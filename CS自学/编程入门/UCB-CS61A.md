# UCB CS61A · Structure and Interpretation of Computer Programs

| 属性 | 值 |
|------|-----|
| 学校 | UC Berkeley |
| 学期 | [2026 Summer](https://cs61a.org/) |
| 语言 | Python 3 |
| 教材 | [Composing Programs](https://www.composingprograms.com/) |
| 总课时 | ~15 周（自学建议 6–10 周，每周 10–15h） |
| 评分 | 4 次 Project + 3 次 Midterm + Lab/Homework |

## 课程目标

用 Python 教授程序构造与解释的核心思想。不是教 Python 语法——是教 ==抽象==：
- 怎么用函数构建抽象屏障
- 怎么用数据组织复杂信息
- 怎么理解程序的执行过程
- 怎么写解释器理解语言的本质
